Privacy Policy

Effective Date: June 1, 2025

Hank Sent Me is a Chrome Extension that helps users score the strength of mutual LinkedIn connections based on publicly visible profile information.

This extension does not collect, store, or transmit any personal information, credentials, or private LinkedIn data. All processing occurs locally within the user browser. No data is sent to any external server or third party.

Hank only analyzes DOM elements visible to any logged-in LinkedIn user and discards all data when the popup is closed.
